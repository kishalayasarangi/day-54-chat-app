# Day 54 — Real Time Chat App (Flask + SocketIO)

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A real time chat app built with Flask and Socket.IO.
Multiple users can chat in real time in different rooms.

## How to run
```bash
pip3 install flask flask-socketio
python3 app.py
```
Then open http://localhost:5000 in multiple tabs!

## Features
- Real time messaging with WebSockets
- Multiple chat rooms
- User join/leave notifications
- Message timestamps
- Online users list
